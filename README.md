## BCCS란?

* 본 시스템은 차량에서 발생한 사고 및 위험 상황을 클라우드에 자동 업로드 함으로써 자료의 유실을 방지하고, 사용자가 해당 정보를 편리하게 사용할 수 있도록 플랫폼을 제공하는 것이 목적이다.
* 주행 및 주차 중 발생할 수 있는 이벤트를 감지한 후, 영상정보를 실시간으로 클라우드에 송신하여 앱에서 이벤트를 받아볼 수 있게 구성하였다.
